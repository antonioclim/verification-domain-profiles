# Release notes 2.0.0

Version 2.0.0 is a major reconstruction of the earlier diagnostic companion package. It introduces the verification-domain profile `Gamma_V(B)`, exact minimum-budget certification, a finite stress dual, harmless/repairable/irreparable domain classes, exact augmented-Chebyshev intervals and structured row-generation oracles.

The release retains 84 minimum-budget certificates, 196 profile records, a frozen computational protocol, raw and processed evidence, four deterministic publication figures and a standalone exact checker.

The release does not claim universal runtime superiority, independent physical-machine replication or exhaustive certification of scholarly priority.
